pyrate.refpixel.ref_pixel(ifgs, params)[source]¶

Returns (y,x) reference pixel coordinate from given ifgs.

If the config file REFX or REFY values are empty or subzero, the search for the reference pixel is performed. If the REFX|Y values are within the bounds of the raster, a search is not performed. REFX|Y values outside the upper bounds cause an exception.

ifgs: list
sequence of interferograms.
params: dict
parameters dictionary
