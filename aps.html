Atmospheric Phase Screen Module¶

This Python module implements a spatio-temporal filter method for correcting interferograms for atmospheric phase screen (APS) signals.

pyrate.aps.spatial_low_pass_filter(ts_lp, ifg, params)[source]¶

Filter time series data spatially using either a Butterworth or Gaussian low pass filter defined by a cut-off distance. If the cut-off distance is defined as zero in the parameters dictionary then it is calculated for each time step using the pyrate.covariance.cvd_from_phase method.

Parameters:
  • ts_lp (ndarray) – Array of time series data, the result of a temporal low pass filter operation. shape (ifg.shape, n_epochs)
  • instance ifg (shared.Ifg) – interferogram object
  • params (dict) – Dictionary of configuration parameters
Returns:

ts_hp: filtered time series data of shape (ifg.shape, n_epochs)
Return type:

ndarray
pyrate.aps.spatio_temporal_filter(tsincr, ifg, params, preread_ifgs)[source]¶

Applies a spatio-temporal filter to remove the atmospheric phase screen (APS) and saves the corrected interferograms. Before performing this step, the time series must be computed using the SVD method. This function then performs temporal and spatial filtering.

Parameters:
  • tsincr (ndarray) – incremental time series array of size (ifg.shape, nepochs-1)
  • ifg (list) – List of pyrate.shared.Ifg class objects.
  • params (dict) – Dictionary of configuration parameter
  • tiles (list) – List of pyrate.shared.Tile class objects
  • preread_ifgs (dict) – Dictionary of shared.PrereadIfg class instances
Returns:

None, corrected interferograms are saved to disk
pyrate.aps.temporal_low_pass_filter(tsincr, epochlist, params)[source]¶

Filter time series data temporally using either a Gaussian, triangular or mean low pass filter defined by a cut-off time period (in years).

Parameters:
  • tsincr (ndarray) – Array of incremental time series data of shape (ifg.shape, n_epochs)
  • epochlist (list) – List of shared.EpochList class instances
  • params (dict) – Dictionary of configuration parameters
Returns:

tsfilt_incr: filtered time series data, shape (ifg.shape, nepochs)
Return type:

ndarray
