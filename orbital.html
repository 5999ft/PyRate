Orbital Error Correction Module¶

This Python module implements residual orbital corrections for interferograms.

exception pyrate.orbital.OrbitalError[source]¶

Generic class for errors in orbital correction.

pyrate.orbital.get_design_matrix(ifg, degree, offset, scale=100.0)[source]¶

Returns simple design matrix with columns for model parameters.

Parameters:
  • ifg – interferogram to base the DM on
  • degree – PLANAR, QUADRATIC or PART_CUBIC
  • offset – True to include offset cols, otherwise False.
  • scale – amount to divide cell size by for improving inversion robustness
pyrate.orbital.get_network_design_matrix(ifgs, degree, offset)[source]¶

Returns larger format design matrix for networked error correction.

The network design matrix includes rows which relate to those of NaN cells. :param ifgs: sequence of interferograms :param degree: PLANAR, QUADRATIC or PART_CUBIC :param offset: True to include offset cols, otherwise False.

pyrate.orbital.independent_correction(ifg, degree, offset, params)[source]¶

Calculates and removes orbital correction from an ifg.

Parameters:
  • ifg – the ifg to remove remove the orbital error from
  • degree – type of model to use PLANAR, QUADRATIC etc
  • offset – boolean
  • params – parameter dictionary
pyrate.orbital.network_correction(ifgs, degree, offset, params, m_ifgs=None, preread_ifgs=None)[source]¶

Calculates orbital correction model, removing this from the ifgs. .. warn:: This will write orbital error corrected phase_data in the ifgs.

Parameters:
  • ifgs – interferograms reduced to a minimum tree from prior MST calculations
  • degree – PLANAR, QUADRATIC or PART_CUBIC
  • offset – True to calculate the model using offsets
  • params – dict, parameter dictionary
  • m_ifgs – multilooked orbfit ifgs (sequence must be mlooked versions of ‘ifgs’ arg)
  • preread_ifgs – dict, parameters dict corresponding to config file
pyrate.orbital.orbital_correction(ifgs_or_ifg_paths, params, mlooked=None, offset=True, preread_ifgs=None)[source]¶

Removes orbital error from given Ifgs.

NB: the ifg data is modified in situ, rather than create intermediate files. The network method assumes the given ifgs have already been reduced to a minimum set from an MST type operation.

Parameters:
  • ifgs – sequence of Ifg objs to correct
  • degree – PLANAR, QUADRATIC or PART_CUBIC
  • method – INDEPENDENT_METHOD or NETWORK_METHOD
  • mlooked – sequence of multilooked ifgs (must correspond to ‘ifgs’ arg)
  • offset (bool) – True/False to include the constant/offset component
