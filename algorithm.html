PyRate Algorithm Module¶

Collection of algorithms used in PyRate.

pyrate.algorithm.get_all_epochs(ifgs)[source]¶

Returns sequence of all master and slave dates in given ifgs.

pyrate.algorithm.get_epochs(ifgs)[source]¶

Returns an EpochList derived from all given interferograms.

pyrate.algorithm.ifg_date_index_lookup(ifgs, date_pair)[source]¶

Returns an Ifg index which has a master/slave dates given in ‘date_pair’. ifgs - list of Ifg objects to search in date_pair - a (datetime.date, datetime.date) tuple

pyrate.algorithm.ifg_date_lookup(ifgs, date_pair)[source]¶

Returns an Ifg which has a master/slave dates given in ‘date_pair’. ifgs - list of Ifg objects to search in date_pair - a (datetime.date, datetime.date) tuple

pyrate.algorithm.is_square(arr)[source]¶
arr: ndarray
numpy array

whether arr is square or not

pyrate.algorithm.least_squares_covariance(A, b, v)[source]¶

Least squares solution in the presence of known covariance.

This function is known as lscov() in MATLAB. A: design matrix b: observations (vector of phase values) v: covariances (weights) in vector form

pyrate.algorithm.los_conversion(phase_data, unit_vec)[source]¶

Converts phase from line-of-sight (LOS) to horizontal/vertical components. phase_data - phase band data array (eg. ifg.phase_data) unit_vec - 3 component sequence, eg. [EW, NS, vertical]

pyrate.algorithm.master_slave_ids(dates)[source]¶

Returns dict of ‘date:unique ID’ for each date in ‘dates’. IDs are ordered from oldest to newest, starting at 0. Replaces ifglist.mas|slvnum in Pirate.

pyrate.algorithm.unit_vector(incidence, azimuth)[source]¶

Returns unit vector tuple (east_west, north_south, vertical).
