pyrate.algorithm.master_slave_ids(dates)[source]¶

Returns a dictionary of ‘date:unique ID’ for each date in ‘dates’. IDs are ordered from oldest to newest, starting at 0. Implements ifglist.mas|slvnum used in the Matlab Pirate package.

Parameters:dates (list) – List of dates
Returns:unique dates IDs
Return type:dict
